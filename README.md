# exercises
User Management API

• GET /user
• POST /user
• PUT /user
• DELETE /user

Pre-Requisite
Java 8 ( >1.8.0_191)

Build and Test 

• Clone from the repo
• Run mvn clean install
• Run mvn clean test
• Run mvn -Dtest=com.uxpsystems.assignment.UserControllerIT test or mvn -Dtest=com.uxpsystems.assignment.UserServiceIT test
